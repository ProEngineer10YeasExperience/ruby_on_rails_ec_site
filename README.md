# Pick Official


It is a web application which trade products selected by pickofficial buyer.

## Requirements

* Ruby 2.5+
* PostgreSQL

## Code Style

* [rubocop](https://github.com/rubocop-hq/rubocop)


## Quick Start

```
$ bundle install
```
Database Setup:

```
$ bin/rails db:create db:migrate db:seed
```
